Featured Projects

A selection of real projects we've built — from business management systems and web applications to creative web experiences. Each one reflects our commitment to clean code, thoughtful design, and reliable performance.

Inventory System
Crochet Business Management System

A purpose-built inventory system for crochet artisans and small craft businesses. The system features a basic but effective inventory module that lets business owners track finished products, customers orders and manage them in real time.

Web Application
Ledger Budget Tracker

A secure web application designed to track financial income and expenses through a centralized analytical dashboard. The system features user authentication, dynamic Chart.js analytics, strict budget limit enforcement, and automated email alerts powered by a web3forums server.

A dedicated showcase website for a student film project built using HTML, CSS, and JavaScript. The platform features a clean user interface to stream the movie, share behind-the-scenes content, and introduce the cast and crew.

Creative Project
Valentine's Day Web Experience

A delightful, animated web experience created as a personal creative project. Features custom animations, interactive elements, and an emotionally engaging design.
